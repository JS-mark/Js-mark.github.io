选择器

^[N],选择嵌套选择器的第个
^[N]表示部分引用，其中 N 是数字(-1, 0, 1 等等)。
^[0]引用嵌套选择器中的第一层，^[1]则引用第一层和第二层。

注：第一层和第二层是一个完整的选择器.foo__bar，但^[0]部分引用第一层，即.foo。
